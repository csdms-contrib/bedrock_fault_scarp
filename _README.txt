
This .zip package contains code and auxiliary files for the 
Bedrock Fault Scarp model.

The files are:

simple_bedrock_fault_scarp.py - main code
model_parameter_dictionary.py - code for reading formatted input
fault_scarp_inputs_gisp2_3.txt - sample input file
gisp2_o18_interp100.txt - input file containing oxygen-18 curve data
bedrock_scarp_x.dat - sample output: point x coordinates at end of run
bedrock_scarp_z.dat - sample output: point z coordinates at end of run

Package assembled and posted to CSDMS by Greg Tucker, May 2011

Read about this model and its uses in:

Tucker, G. E., S. W. McCoy, A. C. Whittaker, G. P. Roberts, 
S. T. Lancaster, and R. Phillips (2011), Geomorphic significance of 
postglacial bedrock scarps on normal-fault footwalls, J. Geophys. Res., 
116, F01022, doi:10.1029/2010JF001861.
